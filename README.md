# Тестовое задание

**Описание:**

Код предоставленный в репозитории полностью не рабочий, кривой и требует срочного рефакторинга. 
По факту это один контроллер для работы с Dadata(https://dadata.ru/api/)

**Задачи:**
1. Сделать код рабочим
2. Поднять докер контейнер с PHP 8.4
3. Рефакторинг
4. Тестирование


**Результат:**

Полностью рабочий код, покрыт тестами, можно поднять в докер контейнере для тестирования.
